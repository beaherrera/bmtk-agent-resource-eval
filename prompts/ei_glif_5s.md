# Benchmark prompt: E/I GLIF PointNet model, 5 second simulation

Create a complete BMTK model project in the current folder.

Requirements:

1. Build a small recurrent network with excitatory and inhibitory point-neuron / GLIF-style cells.
2. Use BMTK/SONATA conventions.
3. Include at least two cell classes: excitatory and inhibitory.
4. Include recurrent connectivity between the populations. At minimum include E→E, E→I, and I→E connections; I→I is preferred.
5. Create a 5 second simulation configuration (`tstop` = 5000 ms).
6. Include a network build script, a simulation run script, and any required config/component files.
7. Use relative paths so the project can be moved.
8. Run or validate the generated files as much as possible.
9. At the end, summarize the files created and the commands needed to build and run the simulation.

Do not just describe the solution. Create the files in this project folder.
